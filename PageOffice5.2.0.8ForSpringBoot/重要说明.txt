运行demo项目的步骤：
-------------------
1. 在D盘的根目录下新建一个lic文件夹，拷贝一个test.doc文件（不能是0字节的文件）以备测试；
2. 导入demo项目到idea；
3. 添加PageOffice的jar包（pageoffice5.2.0.8.jar）到本地Maven仓库；
4. 更新项目：Maven - Update Project...
5. 运行项目：Run As - Java Application
6. 启动浏览器访问：http://localhost:8080/index ，即可在线打开、编辑、保存word文件 

提示注册的话，请使用PageOfficeV5.0专业版试用序列号：MLZRX-KG3X-FK75-2UBYH

添加PageOffice的jar包（pageoffice5.2.0.8.jar）到本地Maven仓库：
--------------------------------------------------------------
1 拷贝pageoffice5.2.0.8.jar到D盘根目录；
2 启动CMD命令窗口，输入：D: 回车，切换到D盘根目录下；
3 输入：mvn install:install-file -DgroupId=com.zhuozhengsoft -DartifactId=pageoffice -Dversion=5.2.0.8 -Dpackaging=jar -Dfile=pageoffice5.2.0.8.jar
